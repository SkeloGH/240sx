Downloading files off MS3 / MS3Pro SD Cards

As captured and stored on the SD card, your data is in a binary format .MS3 file. This file can not be loaded to MegaLogViewer in this format and must be downloaded and translated to an standard msl file. To accomplish this you can use the MS3 SD Console or download directly from the SD card if you have access to it.

Convert MS3 from SD Card


MS3 SD Console (Registered Version Only):

Navigating the MS3 SD Console to download your log files is a fairly easy and straight forward process, you must be connected to the MS3 and before downloading any files the engine must be turned off.

To begin removing the log files from the MS3 / Ms3Pro SD Card , go to
"Browse / Import SD Card"  you will find this under  "Data Logging".
Clicking This will Open the "MS3 SD Consloe"  .  Figure 1.1



1. To download a file from the SD Card first select which files you want,
If you want multiple files you can download them all at once by click and drag 
the mouse over as many files as you want to download . 

2. Once you have selected which files you want
click on the arrow facing down at the upper right hand corner .


3. To up date the log files list from the the SD card click on the refresh 
button in the upper right had corner .





To delete a file from the log list, select which one you wish to delete , then click on the red button with a X in the middle
at the upper right hand corner .You can can also select multiple files to be deleted the same way as described above .







Direct from SD Card (Available on unregistered version):

With an MS3 or if you have access to the SD card you can use the .MS3 files go to the “Data Logging” tab then to
the “Import Conversion” then click on the “Convert Binary Log (MS3,FRD)”.




This will open the "Transform Raw Datalog" dialog .
Select which files you would like to convert , you can select multiples here as well by clicking on one and then hold the
shift key down and click on the last one in the list you want and it will select everything in-between the two you clicked on.


A progress dialog will display and notify you upon completion.
