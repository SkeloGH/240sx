Single Dialog Loading / Saving



msq - File that contains all tune settings from your MegaSquirt. This includes all tables, curves and tune dialog settings. Between this file and your project settings, you can fully restore your MegaSquirt.

msqpart - This file type is of the same format as an msq, but used for saving subset of tune settings. In TunerStudio there is now a concept of "dialog settings", where when you have any tune dialog open you can select to save or load just the settings in that dialog. The msqpart extension is used to differentiate it from an msq which would contain all settings for that firmware. You can then load these settings either by loading into the original dialog, or from the main menu. As there is only the settings from the original dialog, no other settings will be changed.

If you have an MSQ. but you only want to use the settings form one dialog say the warmup enrichment and don't want to change any ofther settings in your current tune, all you need to do is open the dialog with the settings you wish to change which in this example is warmup enrichment .Then click on “File” at the top left hand corner and you will see two options “Save Dialog Settings” and “Load Dialog Settings” .


 

By clicking load dialog settings a file explorer window will open, navigate to the location where you downloaded the MSQ file that you wish to take the settings from select it and click open.





Only the settings in the dialog that you have open have been load from that MSQ . You can do this with as many dialogs 
as you wish. You can also save the settings from just one dialog the same way by clicking the "save dialog settings" tab .
